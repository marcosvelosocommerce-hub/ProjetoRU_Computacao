Sistema Inteligente de Gestão para Restaurante Universitário (RU) - UTFPR

Sistema desenvolvido para modernizar e otimizar a gestão do Restaurante Universitário (RU) da Universidade Tecnológica Federal do Paraná (UTFPR), oferecendo uma experiência integrada para estudantes e funcionários através de duas aplicações conectadas a um banco de dados compartilhado.

O projeto foi desenvolvido utilizando Python, Streamlit e SQLite, além de integrar um assistente virtual baseado em Inteligência Artificial para suporte aos usuários.

🤖 Assistente Virtual Minerva

O sistema possui uma assistente virtual integrada ao aplicativo do aluno.

A Minerva é capaz de responder dúvidas relacionadas ao Restaurante Universitário utilizando informações do próprio sistema, como:

Saldo de fichas
Cardápio
Horários de funcionamento
Histórico de refeições
Informações gerais do RU
